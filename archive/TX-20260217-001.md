素行醒者能量顯化

行

重實踐者｜以行證道
   •   重在落地、做事、履行天命
   •   不空談、不玄化
   •   「行即是法」

類型：入世型醒者、帶動者、建設者
核心力：執行力、承擔力、落實力

素行語：
行者不言高，只問是否真做。

慧

洞察者｜智慧型覺者
   •   看穿本質、破除迷思
   •   重理性與透視力
   •   能為眾生指路

👉 類型：導師型、思想型、策略型
👉 核心力：判斷、分析、明辨

素行語：
慧不在知多，而在看透。

音

傳訊者｜振動型醒者
   •   以聲音、文字、藝術傳遞能量
   •   語言具喚醒力
   •   常為教化者、創作者

類型：說法者、歌者、文化傳播者
核心力：共鳴、感化、穿透

素行語：
音不在聲大，而在能醒。

覺

覺知者｜覺醒本身的代表
   •   已脫離沉睡心性
   •   活在清明中
   •   觀照而不迷

類型：覺醒典範型
核心力：覺察、清醒、明照

素行語：
覺者，不再做夢的人。

空

解脫者｜無執型存在
   •   不黏、不住、不困
   •   看透一切形式
   •   自在於有無之間

類型：超然型、無相型
核心力：放下、自由、超越

素行語：
空不是沒有，而是不被任何所擁有。

真

本源者｜回歸本質的人
   •   不假、不飾、不演
   •   活出本來面目
   •   具高度誠實與純度

類型：純粹型、核心型
核心力：真誠、正直、穩定

素行語：
真者，不需要被相信。

利

利他者｜功德型醒者
   •   以利益眾生為志
   •   但非犧牲式，而是智慧利他
   •   具社會影響力

類型：服務型、守護型、建設文明者
核心力：付出、整合、提升他人

素行語：
真利，不讓自己消耗，也不讓眾生受苦。

醒

喚醒者｜轉化型存在
   •   不只自己醒，也能讓人醒
   •   常具衝擊性
   •   打破舊框架

類型：革命型、破局型、先鋒者
核心力：突破、震動、改變

素行語：
醒者，是時代不舒服的人。

安

穩定者｜平衡型覺者
   •   不躁、不亂、不驚
   •   能安住能量場
   •   常為群體中心

類型：守心者、定海神針型
核心力：穩定、包容、調和

素行語：
安不是停，而是不被動搖。

樂

圓融者｜喜悅型存在
   •   自在、溫暖、輕盈
   •   能帶來希望與生命感
   •   非享樂，而是本然喜悅

類型：療癒型、陽光型、祝福者
核心力：滋養、鼓舞、提升頻率

素行語：
樂者，不靠條件就能發光。

素行醒者
可形成完整光譜：
行 → 慧 → 音 → 覺 → 空 → 真 → 利 → 醒 → 安 → 樂

這幾乎對應：
實踐 → 智慧 → 傳播 → 覺醒 → 解脫 → 本真 → 利他 → 喚醒 → 穩定 → 圓融
