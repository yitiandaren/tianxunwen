# RULE｜Canonical Governance｜原典治理

version: v1_0
date: 2026-05-14

## 系統定位

本系統為：
素行文明原典治理系統。

## 治理層級

RAW → Candidate → Canonical → Published

## 正式內容類型

| Type | 中文 | canonical |
|---|---|---|
| TX | 天訊文 | true |
| LS | 逐字稿母檔 | true |
| EK | 素行經典 | true |
| MN | 素行咒語 | true |
| TS | 主題片段 | false |
| PB | 發布包 | false |
| MA | 媒體資產 | false |

## 核心原則

1. 並非所有內容都 canonicalize
2. RAW 不等於 Canonical
3. AI 不具最終 canonical judgment
4. 原典內容不可改原文
